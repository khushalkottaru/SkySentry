## SkySentry

A performant computer vision system to detetct birds at airports. Built on YOLO11 with SAHI tiled inference and fine-tuned with the AirBirds dataset for enhanced detection in complex airport environments.

Enterprise bird detection systems are expensive and require specialized hardware that make them inacessible to smaller airports. Birdstrikes are the cause of 22,000+ aviation incidents per year, roughly 61 per day. This is a major issue. SkySentry aims to solve this problem by providing a cost-effective, real-time bird detection solution for general aviation airports.

### Goals
- Detect birds in noisy environments with high confidence
- Output actionable risk assessments
- Run on low cost hardware

### Status
Currently in early development.
